# TunkBook - Frontend 📘

Este es el **proyecto frontend** de **TunkBook**, una plataforma web desarrollada con **React**, **Redux** y **React Router**. Se enfoca en la experiencia del usuario, la gestión de estado eficiente y la integración con un backend desarrollado en **Spring Boot** (repositorio separado).

## 🌐 Sobre el Proyecto

Este frontend consume servicios REST del backend para proporcionar una experiencia interactiva y dinámica para autores, lectores y administradores.

## 🌟 Funcionalidades Principales

### 👤 Usuarios

- ✍️ **Autores** pueden:
  - Publicar nuevos libros
  - Gestionar y editar sus libros existentes
  - Crear comunidades
  - Realizar pedidos de libros

### 🛠️ Administrador

- 👥 Gestionar usuarios y permisos
- ❌ Eliminar libros
- 🗑️ Eliminar categorías
- ✏️ Modificar categorías existentes

## 🧱 Tecnologías Utilizadas

- **Framework**: React
- **Enrutamiento**: React Router
- **Gestión de Estado**: Redux
- **Integración Backend**: fetch (REST API)
- **Diseño de Componentes**: Desarrollo modular y reutilizable

## 🎯 Propósito

Este repositorio demuestra habilidades en desarrollo de interfaces modernas, consumo de APIs REST, estructuración de proyectos frontend y buenas prácticas de diseño escalable. Ideal para fines profesionales o búsqueda de empleo.
